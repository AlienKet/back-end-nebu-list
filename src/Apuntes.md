Lenguaje: TypeScript.

Framework: NestJS.

ORM (Modelo de Datos): TypeORM.

Funcionalidad: Permite gestionar la base de datos usando clases de TypeScript (Entidades) en lugar de código SQL manual.

Estructura:

Controllers: Gestionan las rutas y reciben las peticiones de Angular.

DTOs: Validan que los datos que envías desde el front sean correctos (ej: que el status sea uno de los permitidos).

Models (Entities): Definen las tablas de la base de datos (Tareas, Categorías, Usuarios).
